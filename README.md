# Customer Churn Prediction

An end-to-end Machine Learning project for predicting customer churn.

## Features
- Data preprocessing
- Random Forest model
- XGBoost model
- FastAPI REST API
- Streamlit Web App
- Docker support

## Technologies
- Python
- Scikit-learn
- XGBoost
- FastAPI
- Streamlit
- Docker

## Run with Docker

```bash
docker build -t churn-app .
docker run -p 8501:8501 churn-app
